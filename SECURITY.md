# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| `main` (latest deploy) | Yes |
| Older commits | No |

## Reporting a vulnerability

**Do not** open a public GitHub issue for security vulnerabilities.

Email [info@solidsolutions.africa](mailto:info@solidsolutions.africa) with:

- Description of the issue
- Steps to reproduce
- Impact assessment (if known)
- Your contact details for follow-up

We aim to acknowledge reports within **3 business days** and will work with you on a fix before any public disclosure.

## Scope

In scope:

- [solidsolutions.africa](https://solidsolutions.africa) and this repository's deployed build
- GitHub Actions workflows and deployment configuration in this repo

Out of scope:

- Third-party services (Formspree, hosting provider, Telegram, etc.) — report those directly to the vendor
- Social engineering or physical attacks

## Safe harbour

We appreciate responsible disclosure and will not pursue legal action against researchers who report issues in good faith.
